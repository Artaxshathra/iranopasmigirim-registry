Privacy Policy — INRTV Live Extension

Last updated: April 20, 2026

Summary

This extension does not collect, store, or transmit any personal or sensitive user data. All processing happens locally within your browser.

Data Collection

This extension collects no data whatsoever. Specifically:

  • No browsing history is collected or transmitted
  • No IP addresses are logged
  • No analytics or tracking of any kind
  • No user accounts or authentication
  • No cookies are set by the extension
  • No third-party services, SDKs, or analytics tools are used

Local Storage

The extension uses the browser's session storage API to temporarily remember detected stream URLs while the browser is open. This data is stored locally on your device only, is never transmitted anywhere, and is automatically cleared when the browser is closed.

Network Requests

The extension makes network requests only to:

  • iranopasmigirim.com — the official INRTV website (only when you visit it)
  • hls.irannrtv.live — the live stream server (only when you choose to watch)

No data is sent to any other servers. No information about your viewing habits is transmitted.

Permissions

  • storage: Used to temporarily store detected stream URLs in session storage (local only, cleared on browser close)
  • activeTab: Used to detect video streams on the current page when you click the extension icon
  • Host permission (iranopasmigirim.com): Allows the extension to detect the live stream on the official website
  • Host permission (hls.irannrtv.live): Allows the player to load the live video stream

Third-Party Code

The extension bundles hls.js (MIT license) for HLS video playback. This library runs entirely within your browser and makes no external requests beyond loading the video stream segments you choose to watch.

Changes

If this policy is ever updated, changes will be noted here with a revised date.

Contact

For questions about this privacy policy, open an issue on the project's GitHub repository.
