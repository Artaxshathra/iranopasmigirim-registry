Privacy Policy — INRTV Live Extension

Last updated: April 20, 2026

Summary

This extension does not collect, store, or transmit any personal or sensitive user data. All processing happens locally within your browser.

Data Collection

This extension collects no data whatsoever. Specifically:

  • No browsing history is collected or transmitted
  • No IP addresses are logged by the extension
  • No analytics or tracking of any kind
  • No user accounts or authentication
  • No cookies are set by the extension
  • No local storage, session storage, or any browser storage APIs are used
  • No third-party services, SDKs, or analytics tools are included

Permissions

The extension requests no browser permissions (no storage, no activeTab, no tabs). The only declaration is:

  • Host permission (https://hls.irannrtv.live/*): Allows the built-in player to load the live video stream from the INRTV streaming server. This is the only network destination the extension contacts.

Network Requests

The extension makes network requests to exactly one domain:

  • hls.irannrtv.live — the INRTV live stream server, contacted only when you click "Watch Live" to start playback.

The popup contains a link to iranopasmigirim.com (the official INRTV website). Clicking this link opens the site in a normal browser tab — this is a standard navigation, not an extension-initiated network request. No data is sent to this domain by the extension.

Third-Party Code

The extension bundles hls.js (MIT license) for HLS video playback. This library runs entirely within your browser and makes no external requests beyond loading the video stream segments from hls.irannrtv.live when you choose to watch.

Changes

If this policy is ever updated, changes will be noted here with a revised date.
